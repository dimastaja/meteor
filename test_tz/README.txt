тестовое задание для веб разработчика:
есть верстка главной страницы сайта (см папку Trablz), логика расположения блоков при отображении должна быть такова, что бы на сайте не было пустых мест на любом разрешении ( хоть 4k ). у каждого блока есть минимальные размеры ( должны быть возможность их указывать в конфиге/коде ), и увеличение размера блоков - только пропорционально. задача - написать на io.js скрипт, который каким либо способом должен получить размер экрана пользователя, и сгенерировать расположение блоков под это разрешение так, что бы не было пустых пятен. (кол-во блоков - не ограничено)
